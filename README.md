# thunar_custom-actions
A collection of self-developed custom actions for the xfce file browser thunar

**Convert selection to JPEG**

Select a bunch of images, right click and choose "Convert selection to JPEG". This converts all selected images to JPEG. The selcted images will not be deleted.

**Convert selection to MP3**

Select *.ogg,*.flac,*.wv,*.wav,*.mpc,*.ape,*.shn or*.wma audio files, right click and choose "Convert selection to MP3". This converts the selected audio files to MP3.
